Добро Пожаловать!


Это учебный проект предназначенный для помощи начинающим специалистам своей первой работы. Те из нас, кто уже сталкивался с задачей найти первую работу в новой сфере, знают что это задача не из самых легких, десятки отправленных резюме и не одного ответа

Задача сервиса помочь соискателю с этим не легким трудом, автоматизировать процесс поиска работы. Сервис по настроинным пользователем фильтрам, будет рассылать указанное резюме пользователя работодателям, выложившим свои обьявления на платформе Head hunter, собирая статиску по оставленным откликам

Приложение не хранит пароли пользователя, вся авторизация осуществляется, через ресурс Head hunter.
